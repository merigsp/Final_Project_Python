# Battleship ️

## Description
This is a simplified version of the classic Battleship game in Python.  
The game runs entirely in the terminal.

---

## Project Structure

├─ main.py # Entry point
├─ src/
│ ├─ ship_input.py # Player ship placement
│ ├─ bot_generation.py # Bot ship generation
│ ├─ gameplay.py # Game logic and loop
│ └─ utils.py # Helper functions
├─ data/ # CSV files for ships and game state
├─ outputs/ # Logs or additional files (empty)
├─ requirements.txt # Project dependencies
└─ README.md # This file

## How to Run

1. Install dependencies

pip install -r requirements.txt

2. Run the game

python3 -m main
