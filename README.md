# Battleship ️

## Description
This is a simplified version of the classic Battleship game in Python.  
The game runs entirely in the terminal.

---

## Project Structure

├─ main.py
├─ data/
│ ├─ player_ships.csv 
│ ├─ bot_ships.csv 
│ ├─ game_state.csv
├─ src/
│ ├─ ship_input.py
│ ├─ bot_generation.py
│ ├─ gameplay.py
│ └─ utils.py
├─ outputs/
│ └(any logs or additional result files)
├─ requirements.txt
└─ README.md

## How to Run

1. Install dependencies

pip install -r requirements.txt

2. Run the game

python3 -m main
